# Awesome Artale
Artale 實用工具精選

## 關於 Artale
Artale 是一款建立在 MapleStory Worlds 平台上的多人 RPG 世界，主打重現經典《楓之谷》的懷舊體驗，讓玩家能再次踏上熟悉的冒險旅程。
> [!TIP]
> 本精選將著重在支援 [Artale (繁體中文版)](https://maplestoryworlds.nexon.com/zh-tw/play/cabbc2ce91844c79989b18931fe492fe/) 的工具上。

### 計算機
- [Artale裝備效益計算機](https://jeremyabcd.github.io/ArtaleCalculate/)
- [Artale工具集](https://isaswa.github.io/artale-tool-sets/)
    - [經驗值時間效率計算機](https://isaswa.github.io/artale-tool-sets/artale-exp-time-calculator/) - 計算練功所需時間，支援經驗加成、排程規劃與等級歷史記錄。

    - [裝備成本計算機](https://isaswa.github.io/artale-tool-sets/artale-gear-cost-calculator/) - 比較裝備替換方案，計算每單位表攻提升所需成本。

    - [經驗值計算機](https://artale-tools.archie-design.app/tools/exp-calculator) - 計算升級所需時間與經驗值，規劃你的練功路線。
    - [組隊經驗值計算機](https://artale-tools.archie-design.app/tools/party-exp-calculator) - 計算組隊時的經驗值分配與效率，找到最佳組隊策略。    
    - [祭司算血計算器](https://artale-tools.archie-design.app/tools/priest-hp-calculator) - 計算面對怪物攻擊所需的魔防與HP，規劃你的生存策略。
    - [衝卷計算機](https://artale-tools.archie-design.app/tools/scroll-calculator) - 計算不同成功率捲軸的期望值，幫助你做出最佳投資決策。
    - [命中率計算機](https://artale-tools.archie-design.app/tools/accuracy-calculator) - 計算不同職業對怪物的命中機率，優化你的戰鬥配置。
